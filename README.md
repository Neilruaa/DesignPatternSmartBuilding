# Smart Building Surveillance System

Système de surveillance de bâtiment en TypeScript utilisant 4 design patterns : 

**Adapter** pour utiliser des API aux output différentes (Marque B).

**Abstract Factory** pour créer des instances de capteurs sans être bloqué en termes de nombres de marques différentes de capteurs.

**Observer** pour que plusieurs instances de Notifications puisse avertir d'un seul changement.

**Iterator**. utiliser plusieurs fois, par exemple pour parcourir touts les capteurs d'une pièce.

## Lancer le projet

```bash
npm i
npm start
```
